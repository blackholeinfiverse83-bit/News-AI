## ğŸš€ Getting Started

Follow these simple steps to get the project running on your local machine.

### Prerequisites

- **Node.js**: v18 or higher
- **Python**: v3.9 or higher
- **Git**

### 1. Clone the Repository

```bash
git clone <REPO_URL> "News AI"
cd "News AI"
```

### 2. Backend Setup

Navigate to the backend directory and set up the Python environment.

```bash
cd unified_tools_backend

# Create virtual environment
python -m venv .venv

# Activate virtual environment
# Windows:
.venv\Scripts\activate
# Mac/Linux:
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

**Configuration**: Create a `.env` file in `unified_tools_backend/` with your API keys (OpenAI, Grok, Serper, YouTube, etc.).

### 3. Frontend Setup

Open a new terminal, navigate to the frontend directory, and install dependencies.

```bash
cd blackhole-frontend

# Install Node modules
npm install
```

**Configuration**: Create a `.env.local` file in `blackhole-frontend/` (if needed) to override defaults. By default, it connects to `http://localhost:8000`.

## â–¶ï¸ Usage

### Start the Backend

In your backend terminal:
```bash
# Ensure venv is active
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
- API Docs: [http://localhost:8000/docs](http://localhost:8000/docs)

### Start the Frontend

In your frontend terminal:
```bash
npm run dev
```
- UI: [http://localhost:3000](http://localhost:3000)

### How to Use
1. Open the frontend at `http://localhost:3000`.
2. Enter a valid news URL in the input box.
3. Watch the **Live Pipeline** process the article in real-time.
4. View the result: Credibility score, Summary, and Related Videos.
5. Provide feedback (Like/Skip) to help the AI learn.
